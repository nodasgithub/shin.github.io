Project: Happiness Jar
Language: Java
Software: Android
Summary:
  • This app works as a notepad, and users can take a note for their happy things.
  • Used SQLite to store data locally on the phone.
  • There are three functions for this app: create, edit, and remenber.
  • The create function make users be available to create notes.
  • By using the edit function, users can edit notes.
  • The remember function displays a happy things randomly from users' note.
