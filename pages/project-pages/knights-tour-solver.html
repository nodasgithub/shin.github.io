Project: Knight's Tour Solver
Language: Java (Maven, JavaFX)
Software: Eclipse
Summary:
  • This application solves the knight tour problem.
  • I used recursion, the back tracking, and heuristics to solve the problem.
  • This application find answers for all 8 * 8 positions.
  • To successfully find solutions, the knight prioritizes to move to positions where have a few choices for the next move like corners or edges.
