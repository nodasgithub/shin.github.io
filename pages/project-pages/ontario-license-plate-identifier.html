Project: Ontario License Plate Identifier
Language: Java (Maven, JavaFX)
Software: NetBeans
Summary:
  • This application analyze and conver the Ontario Driver's License image into String.
  • Coded from scratch, and it works without other external resources.
  • In the backgroud, the app divides images into some quadrants and analyzes each word.
  • During the analysis, the image will convert into two colours, white and blue.
  • The format of the plate should be WWWW-NNN(W: word, N: number). e.g. AAAA-000.
  • If users want, they can restore new data in order for the app to make it more accurate. (The learning function)
